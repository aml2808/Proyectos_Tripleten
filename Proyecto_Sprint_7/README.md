## vehicles-web-app
Aplicacion web construida con streamlit para explorar un conjunto de datos
de anuncios de venta de vehiculos en Estados Unidos.

## Proposito del proyecto
Construir una aplicacion web que permite explorar un conjunto de datos 
de anuncios de venta de vehiculos por medio de graficos (histograma y grafico de dispersion).
Se emplearon diferentes herramientas durante la puesta en marcha de la aplicacion:
- Anaconda prompt
- Jupyter Notebook
- VS code
- GitHub
- Render

## Funcionalidades
- Visualizacion de un histograma del odometro
- Grafico de dispersion entre precio y odometro
La aplicacion genera graficos interactivos.

## Instrucciones de uso
- Accede a la aplicacion a traves del enlace: https://vehicles-web-app-t6h1.onrender.com
- Utiliza los botones en la interfaz para mostrar los diferentes tipos de grafico.
- Analiza los graficos generados para identificar patrones y tendencias del conjunto de datos.
